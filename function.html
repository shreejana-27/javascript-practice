function is a block of code to perform the particular task. It can be reuse many times, 
        so it is necessary for large application.
        we always call the function using parenthesis() except in specific condition
        eg. myFunction()

        we will define the function like this:

        function functionName(){
            ...code goes here..
        }

        in modern javascript we will use arrow function to define function as follows:

        function functionName=() => {
            ...code goes here..
        }
